An Anki add-on that allows to create a filtered deck for each real deck you have and for each deck and some search term. E.g. this allows you to study all due cards in the order created and if you sync after deck creation to mobile also on mobile. I would only do this with the "Anki 2.1 scheduler (beta)" because the default scheduler resets cards in learning when you empty or rebuild a filterd deck. This might also be useful if you want to sort all overdue cards by their overdueness per deck, see e.g. the Anking video Anki:How to Miss a Day (making up skipped days or studying ahead without a penalty) or here.
